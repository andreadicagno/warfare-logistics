# OpenFrontIO - Fonte di Ispirazione

**URL**: https://github.com/openfrontio/OpenFrontIO

## Cos'è

OpenFrontIO è un RTS browser-based focalizzato sul controllo territoriale e la costruzione di alleanze. È una riscrittura di WarFront.io, sviluppato in TypeScript con architettura client-server.

## Caratteristiche di Gameplay Rilevanti

1. **Controllo Territoriale**: Espansione del territorio su mappe basate sulla geografia reale
2. **Sistema di Alleanze**: Formazione di alleanze per difesa reciproca
3. **Gestione Risorse**: Bilanciamento tra espansione e capacità difensive
4. **Multiplayer Real-time**: Gameplay simultaneo

## Architettura

- `/src/client` - Frontend
- `/src/core` - Logica condivisa
- `/src/server` - Backend
- `/resources` - Asset statici (mappe, immagini)

## Spunti per il Nostro Progetto

- Meccaniche multi-layer: territorio, risorse, alleanze interconnessi
- Mappe permanenti con influenza geopolitica
- Dinamiche collaborative tramite alleanze
- Architettura scalabile per multiplayer
